### GB/T 2440-2017 尿素

**监管分类**：农业生产资料 → 肥料

| 技术指标名称 | 约束 | 指标值 | 单位 |
|:-----------|:---:|:-----:|:----:|
| 总氮(N)的质量分数（农业用优等品） | ≥ | 46.0 | % |
| 总氮(N)的质量分数（农业用合格品） | ≥ | 45.0 | % |
| 缩二脲的质量分数（农业用优等品） | ≤ | 0.9 | % |
| 缩二脲的质量分数（农业用合格品） | ≤ | 1.5 | % |
| 水分（农业用优等品） | ≤ | 0.5 | % |
| 水分（农业用合格品） | ≤ | 1.0 | % |
| 粒度0.85mm~2.80mm（农业用优等品） | ≥ | 93 | % |
| 粒度0.85mm~2.80mm（农业用合格品） | ≥ | 90 | % |
| 总氮(N)的质量分数（工业用优等品） | ≥ | 46.4 | % |
| 总氮(N)的质量分数（工业用合格品） | ≥ | 46.0 | % |
| 缩二脲的质量分数（工业用优等品） | ≤ | 0.5 | % |
| 缩二脲的质量分数（工业用合格品） | ≤ | 1.0 | % |
| 水分（工业用优等品） | ≤ | 0.3 | % |
| 水分（工业用合格品） | ≤ | 0.7 | % |
| 铁(Fe)的质量分数（工业用优等品） | ≤ | 0.0005 | % |
| 铁(Fe)的质量分数（工业用合格品） | ≤ | 0.0010 | % |
| 碱度(以NH₃计)（工业用优等品） | ≤ | 0.01 | % |
| 碱度(以NH₃计)（工业用合格品） | ≤ | 0.03 | % |
| 硫酸盐(以SO₄计)（工业用优等品） | ≤ | 0.005 | % |
| 硫酸盐(以SO₄计)（工业用合格品） | ≤ | 0.020 | % |
| 水不溶物（工业用优等品） | ≤ | 0.005 | % |
| 水不溶物（工业用合格品） | ≤ | 0.040 | % |
