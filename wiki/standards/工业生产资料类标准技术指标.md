# 工业生产资料类标准技术指标

> 所属监管目录：[[wiki/concepts/工业生产资料]]
> 共 **9份** 标准，**40项** 技术指标

## 水泥

> 相关概念：[[wiki/concepts/水泥]]

### GB 175-2023 通用硅酸盐水泥

**监管分类**：工业生产资料 → 水泥

| 技术指标名称 | 约束 | 指标值 | 单位 |
|:-----------|:---:|:-----:|:----:|
| 3d抗压强度(P·O 42.5) | ≥ | 17.0 | MPa |
| 3d抗压强度(P·O 52.5) | ≥ | 23.0 | MPa |
| 28d抗压强度(P·O 42.5) | ≥ | 42.5 | MPa |
| 28d抗压强度(P·O 52.5) | ≥ | 52.5 | MPa |
| 初凝时间 | ≥ | 45 | min |
| 终凝时间 | ≤ | 600 | min |
| 氧化镁(MgO) | ≤ | 5.0 | % |
| 三氧化硫(SO₃) | ≤ | 3.5 | % |
| 氯离子(Cl⁻) | ≤ | 0.06 | % |

### GB/T 748-2023 抗硫酸盐硅酸盐水泥

**监管分类**：工业生产资料 → 水泥

| 技术指标名称 | 约束 | 指标值 | 单位 |
|:-----------|:---:|:-----:|:----:|
| C₃A含量（中抗硫酸盐） | ≤ | 5.0 | % |
| C₃A含量（高抗硫酸盐） | ≤ | 3.0 | % |
| C₃S含量（高抗硫酸盐） | ≤ | 50.0 | % |
| 14d线膨胀率 | ≤ | 0.060 | % |
| 抗硫酸盐侵蚀系数(K) | ≥ | 0.80 |  |

## 化工原料

> 相关概念：[[wiki/concepts/national-standards-gb]]

### GB/T 320-2025 工业用合成盐酸

**监管分类**：工业生产资料 → 化工原料

| 技术指标名称 | 约束 | 指标值 | 单位 |
|:-----------|:---:|:-----:|:----:|
| 总酸度(以HCl计)（优等品） | ≥ | 31.0 | % |
| 总酸度(以HCl计)（合格品） | ≥ | 31.0 | % |
| 铁(以Fe计)（优等品） | ≤ | 0.002 | % |
| 铁(以Fe计)（合格品） | ≤ | 0.008 | % |
| 灼烧残渣（优等品） | ≤ | 0.005 | % |
| 灼烧残渣（合格品） | ≤ | 0.010 | % |
| 游离氯(以Cl计)（优等品） | ≤ | 0.002 | % |
| 游离氯(以Cl计)（合格品） | ≤ | 0.005 | % |

### GB/T 5138-2021 工业用液氯

**监管分类**：工业生产资料 → 化工原料

| 技术指标名称 | 约束 | 指标值 | 单位 |
|:-----------|:---:|:-----:|:----:|
| 氯含量(体积分数)（优等品） | ≥ | 99.8 | % |
| 氯含量(体积分数)（合格品） | ≥ | 99.6 | % |
| 水分含量（优等品） | ≤ | 0.01 | % |
| 水分含量（合格品） | ≤ | 0.03 | % |

### GB 19106-2013 次氯酸钠

| 游离碱(以NaOH计) | = | 0.1~1.0 | % |

### GB/T 338-2025 工业用甲醇

**监管分类**：工业生产资料 → 化工原料
