# 燃气与能源类国家标准技术指标

> 所属领域：[[wiki/concepts/液化石油气（LPG）]]、[[wiki/concepts/液化天然气（LNG）]]

## GB 11174-2025 液化石油气

| 技术指标 | 约束 | 指标值 | 单位 |
|---------|:----:|:-----:|:----:|
| 蒸气压(37.8℃) | ≤ | 1380 | kPa |
| C₅及C₅以上组分含量 | ≤ | 3.0 | %(体积分数) |
| 总硫含量 | ≤ | 343 | mg/m³ |
| 硫化氢 | ≤ | 10 | mg/m³ |
| 游离水 | 无 | — | — |

## GB/T 23799-2021 车用甲醇汽油(M85)

| 技术指标 | 约束 | 指标值 | 单位 |
|---------|:----:|:-----:|:----:|
| 甲醇含量 | = | 84~86 | %(体积分数) |
| 实际胶质 | ≤ | 5 | mg/100mL |

## GB/T 33445-2023 煤制合成天然气

| 技术指标 | 约束 | 指标值 | 单位 |
|---------|:----:|:-----:|:----:|
| 高位发热量 | ≥ | 34.0 | MJ/m³ |
| 总硫(以硫计) | ≤ | 1.0 | mg/m³ |
| 硫化氢 | ≤ | 0.1 | mg/m³ |
