# 肥料类国家标准技术指标

> 所属领域：[[wiki/concepts/肥料]]

## GB/T 2440-2017 尿素

**农业用（肥料）尿素**

| 技术指标 | 约束 | 优等品 | 合格品 | 单位 |
|---------|:----:|:-----:|:-----:|:----:|
| 总氮(N)的质量分数 | ≥ | 46.0 | 45.0 | % |
| 缩二脲的质量分数 | ≤ | 0.9 | 1.5 | % |
| 水分 | ≤ | 0.5 | 1.0 | % |
| 粒度(0.85mm~2.80mm) | ≥ | 93 | 90 | % |

**工业用尿素**

| 技术指标 | 约束 | 优等品 | 合格品 | 单位 |
|---------|:----:|:-----:|:-----:|:----:|
| 总氮(N)的质量分数 | ≥ | 46.4 | 46.0 | % |
| 缩二脲的质量分数 | ≤ | 0.5 | 1.0 | % |
| 水分 | ≤ | 0.3 | 0.7 | % |
| 铁(以Fe计)的质量分数 | ≤ | 0.0005 | 0.0010 | % |
| 碱度(以NH₃计) | ≤ | 0.01 | 0.03 | % |
| 硫酸盐(以SO₄计) | ≤ | 0.005 | 0.020 | % |
| 水不溶物 | ≤ | 0.005 | 0.040 | % |
